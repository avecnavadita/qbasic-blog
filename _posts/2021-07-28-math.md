## Program For Addition

```
REM "-----------------------------------------"
REM "This program is used to add two numbers"
REM "-----------------------------------------"


INPUT "Enter the first number"; num1
INPUT "Enter the second number"; num2

sum = num1 + num2
REM Print "the sum of"; num1; "and"; num2; "is"; sum
PRINT num1; "+"; num2; "="; sum

END
```

## Program to Subtract

```
REM -----------------------------------------
```
